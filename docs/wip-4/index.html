Memory Operating Profit Monitor

메모리 반도체 업체들의 영업이익 증가율 추이를 먼저 보고, 매출·영업이익·마진을 표로 비교합니다.

무엇을 보나?

  • 메모리 사이클은 영업이익 증가율이 먼저 크게 움직이는 경우가 많습니다.
  • 그래프는 최근 최소 4개 분기의 영업이익 전분기 대비 증가율을 회사별로 비교합니다.
  • 표에서는 매출액과 성장률, 영업이익과 이익률, 전분기 대비 증가율을 함께 봅니다.

메모리 업체 영업이익 흐름

TSMC, Kioxia, SanDisk, Micron, Samsung Electronics, SK Hynix 중 공개 분기 데이터가 있는 회사를 표시합니다.

메모리 실적 데이터를 불러오는 중입니다.

영업이익 증가율 추이

최근 분기별 QoQ 기준입니다. 적자 전환·흑자 전환 구간은 변동률이 크게 보일 수 있습니다.

매출과 영업이익 비교표

최근 분기 기준. 매출 성장률과 영업이익 증가율은 전분기 대비입니다.
