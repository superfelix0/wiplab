Memory Operating Profit Monitor

메모리 반도체 업체들의 분기별 영업이익 증가율과 실적 흐름을 비교합니다.

무엇을 보나?

  • 영업이익 증가율은 이번 분기 영업이익이 직전 분기보다 얼마나 늘거나 줄었는지를 보여줍니다.
  • 기본 그래프는 회사별 묶음 안에 최근 결산 분기 포함 4개 분기를 얇은 막대로 나열합니다.
  • 키옥시아는 비교 가능한 공개 분기 데이터 범위가 부족해 이 페이지에서는 제외합니다.

메모리 업체 수익성 흐름

TSMC, SanDisk, Micron, Samsung Electronics, SK Hynix의 공개 분기 데이터를 기준으로 비교합니다.

메모리 실적 데이터를 불러오는 중입니다.

영업이익 증가율 막대그래프

각 회사의 최근 4개 분기 영업이익 증가율입니다. 예를 들어 2026년 1분기는 2025년 4분기 대비 증가율로 계산합니다.

영업이익 증가율 4분기

최근 결산 분기를 포함한 4개 분기의 전분기 대비 영업이익 증가율입니다. 흑자 전환이나 적자 전환 구간은 변동률이 크게 보일 수 있습니다.

매출과 영업이익 비교표

최근 결산 분기 기준입니다. 하단 표는 매출액, 매출 성장률, 영업이익, 영업이익률만 간단히 비교합니다.
