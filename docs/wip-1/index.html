네이버증권 SKHY.O ADR, 네이버증권 코스피 000660, 네이버 환율 USD/KRW를 자동으로 불러와 같은 원화 기준으로 비교합니다. 계산 기준은 ADR 환산 본주가 = ADR 가격 × 환율 ÷ 0.1입니다. ADR 1주가 본주 1/10을 대표하므로 본주 1주 기준 가격은 ADR 원화가의 10배로 환산합니다. 한국장과 미국장은 정규 거래 시간이 겹치지 않으므로, 각 가격의 기준 시각이 다를 수 있습니다.

HOW TO READ

해석 방법

  1. 괴리율이 플러스면 ADR 환산 본주가가 코스피 본주 가격보다 높다는 뜻입니다.
  2. 괴리율이 마이너스면 ADR 환산 본주가가 코스피 본주 가격보다 낮다는 뜻입니다.
  3. 실제 거래 전에는 시세 지연, 환전 스프레드, 세금, 거래 수수료, 유동성, 운영 시간 차이를 함께 봐야 합니다.
  4. 세션 불일치가 표시되면 ADR, 본주, 환율의 기준 시각이 다르다는 뜻입니다. 이때의 괴리율은 실시간 차익거래 신호가 아니라 참고용 비교값입니다.
