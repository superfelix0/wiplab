KOSPI PER Monitor

A quick valuation temperature check using historical average PER, current KRX PER, and a forward PER reference.

What to read

  • PER shows how expensive the market is relative to current earnings.
  • The historical average gives a long-run reference point.
  • Forward PER can imply undervaluation, but it also depends on confidence in future earnings.
